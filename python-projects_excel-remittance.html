Description:

This Python script automates the validation and preparation of remittance data received from a provider. The provider submits multiple Excel files representing prescription claims. These files collectively support a payment (posted as a single credit in SAP). The script ensures all claims are present and aligned with the invoice and generates a reference file for automated clearing in SAP.

Business Scenario

In the SAP environment, providers may submit one lump-sum payment (credit) that corresponds to multiple individual claims. These claims arrive as Excel files (one per store or batch), typically named with a transaction date.
