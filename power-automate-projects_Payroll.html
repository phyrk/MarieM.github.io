Power Automate Projects

Automation workflows built with Power Automate to streamline business processes across Microsoft 365 and other platforms.

Description:

TTo streamline a time-consuming and error-prone process, I built a Power Automate flow that automatically collects and organizes technician timesheets submitted via email. What was previously a manual task shared between three team members — each spending hours every week — is now a fully automated workflow integrated with SharePoint and Power Query, allowing a single person to complete the entire process in minutes.

Business Problem

Once a week, our field technicians submit their timesheets (~200+ files) via email to a shared Outlook mailbox. Each file contains detailed hour breakdowns for:

  • ✅ Regular Hours
  • ✅ Overtime Hours
  • ✅ Double Time
  • ✅ Lead Hand Hours
  • ✅ Gas Ticket Hours
  • ✅ Refrigeration/Electrical Hours
  • ✅ Night Shift Premium

    • Payroll Flow

    I built a Power Automate flow that completely eliminated the manual collection step and streamlined the entire process.

  • Trigger The flow starts when a new email arrives in the shared mailbox when a new email arrives in the shared mailbox (When a new email arrives (V3) trigger).
  • Attachments:For each incoming email, it checks for attachments (Excel or CSV timesheets)
  • Storage:All attachments are automatically saved to SharePoint in a centralized folder, using the Create file action inside an Apply to each loop
  • Reporting:The data is then loaded into a Power Query-powered report, which:
    • Merges all timesheets
    • Applies pay rates based on role/region
    • Outputs a clean compensation summary

    Result

    • ✅ Time saved: Reduced from ~10 hours/week to just 15 minutes
    • ✅ Scalability: Works for any number of incoming timesheets
    • ✅ Simplicity: One team member can now complete the entire weekly process without manual work
    • ✅ Accuracy: Removes copy-paste errors and ensures consistent calculations

      • Tools Used

      • Power Automate (Outlook, SharePoint)
      • Power Query (Excel)
      • Shared Mailbox
      • Dynamic file handling
